CSS单位 px|em|rem 的区别

Posted on 2019-12-23 In 开发

CSS units for font-size: px | em | rem

CSS中字体大小单位很多，最常用的为px、em以及rem.

px

px即Pixels，可以说是最简单的一个单位。但有一个关键的问题：用户更改了浏览器的默认字体大小，却反映不出想要的结果。

em

它的值不是固定的，一个单位的em等于它父级元素字体的大小。按照浏览器的默认字体16px，那么1em = 16px.
这里有个例子。


在这个例子中，.parent的font-size:20px，.child的font-size=1.5em.
于是，#outerChild的大小为30px，#innerChild的大小为45px(可以嵌套的).

rem

rem的值与root html element(根html元素)有关，和它的父级元素没有关系。如果保持默认字体大小16px不变font-size: 100%，那么1rem=16px。但是这会使我们之后的计算不舒服，很难和px对应，比如10px=0.625rem.
解决方式为在根元素html中加入font-size: 62.5%,这会使得1rem=10px
