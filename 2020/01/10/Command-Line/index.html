关于命令行

Understanding the Command Line

在Mac上,我们将使用Terminal程序,在其中我们可以运行Bash Shell.
在理解Bash Shell之前,我们先联想瓜子皮(Shell)和瓜子仁(Kernel),在我们的计算机系统中,真正核心的就是Kernel,而Shell则使得用户能与硬件交互(interface).这里有两种Shell:GUI(Graphical User Interface)如Finder,CLI(Command Line Interface)如Terminal.
回到Bash Shell, Bash(Bourne Again Shell),是一个命令行解释器(command line interpreter),位于unix或类unix系统中.
实际上,很多情况下我们可以使用GUI代替CLI,那为什么还要用Command Line呢? 其实是为了更好的控制(greater control),以及更有效率,比如我们想新建一个Music文件夹,只需输入mkdir Music.

Command Line Techniques and Directory Navigation

ls list all the items inside the folder
cd change directory
cd ~ go back to root directory
cd .. go back to father directory
clear clear the entire screen
pwd print working directory

ctrl + A go to the beginning of the line
ctrl + E go to the end of the line
ctrl + U clear this line

CheatSheet

Creating,Opening and Removing Files

mkdir NAME make directory
touch test.txt create new file
open test.txt open file
rm test.txt remove file
rm * remove all files
re -r remove entire directory
