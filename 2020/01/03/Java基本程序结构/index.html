Integer Types

整型用于没有小数部分的数值，允许是负数。Java提供了四种整形：

Type Storage Requirement Range (Inclusive)
int 4 bytes –2,147,483,648 to 2,147,483,647 (just over 2 billion)
short 2 bytes –32,768 to 32,767
long 8 bytes –9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
byte 1 byte –128 to 127

通常我们使用int类型。

Floating-Point Types

Type Storage Requirement Range
float 4 bytes Approximately ±3.40282347E+38F (6–7 significant decimal digits)
double 8 bytes Approximately ±1.79769313486231570E+308 (15 significant decimal digits)

这里我们常用double（也被称为双精度数值）。
若要使用float类型，需在数值后加后缀f/F，如3.14F。没有后缀则默认为double类型。

The char Type

char类型本来用于表示单个字符，后来（Unicode字符超过了65536，即2^16个）16位的char类型不够用了，一些Unicode字符需要两个char值。
char类型的字面量值（Literal value）要用单引号括起来。如’A’是编码值为65所对应的字符常量。char类型的值可以表示为十六进制值，范围从\u0000到\Ufff.如\u03C0表示π。
此外还有一些表示特殊字符的转义序列：

Escape Sequence Name Unicode Value
\b Backspace \u0008
\t Tab \u0009
\n Linefeed \u000a
\r Carriage return \u000d
" Double quote \u0022
' Single quote \u0027
\ Backslash \u005c

The boolean Type

布尔类型有两个值：false和true，用来判定逻辑条件。注意Java中整型值和布尔值之间不能进行相互转换，而c++中数值甚至指针可以代替boolean值，值0相当于false。
